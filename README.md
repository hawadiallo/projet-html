# TP_docker_web

### Step 1 : 

> Install Nodejs : https://nodejs.org/en/ </li>

### Step 2 : 

> With the terminal, go to the folder and run the command :

    npm install
   
### Step 3 : 

> When the action is finished, launch the server with the command : 

    node bin/www
  
### Step 4 : 

> Go to the URL : localhost:8080
